### Step 1: Create a Facebook Developer Account

To access the WhatsApp Business API, you'll first need a Facebook Developer account.

1. Go to the [Facebook Developer Portal](https://developers.facebook.com/).
2. Click **Get Started** or **My Apps**.
3. If you don't have an account, click **Sign Up** and follow the instructions.
4. Go to **My Apps** > **Create App**.

### Step 6: Add a System User & Generate a Permanent Token

1. Open [Meta Business Settings](https://business.facebook.com/settings).
2. In the left sidebar, go to **Users > System Users**.
3. Follow the steps shown below, to add a system user.
4. Go back to **Business Settings > System Users**.
5. Select the system user you created.
6. Under **Assigned Assets**, click **Add Assets**, and give access to the app and WhatsApp account.
7. Then under the system user, click **Generate Token**.
8. Choose the app you created and select the necessary permissions.
9. Click **Generate Token**.

> üõ°Ô∏è **Note:** This token can be used as a long-lived token. Store it securely.

### Step 7: Register Your Mobile Number via Postman

1. Go to the **Facebook Developer Platform** and navigate to:  
   `WhatsApp > API Setup`

2. Copy the **Phone Number ID** shown there.

3. Open **Postman** and create a **POST** request with the following details:

   - **URL**  
     ```
     https://graph.facebook.com/v18.0/{PHONE_NUMBER_ID}/register
     ```

   - **Headers**
     ```
     Authorization: Bearer ACCESS_TOKEN
     Content-Type: application/json
     ```

   - **Body**
     ```json
     {
       "messaging_product": "whatsapp",
       "pin": "YOUR_OTP_CODE"
     }
     ```

   > Replace `ACCESS_TOKEN` with the token generated for your system user (from the previous step).  
   > Replace `YOUR_OTP_CODE` with the OTP you received during number verification.

4. Click **Send**.  
   If the response contains `"success": true`, your number will show as **Connected** in **Meta Business Suite** under the WhatsApp section.

### Step 8: Set Up Webhooks

1. Go to **Configuration** under the **WhatsApp** product.
2. Click **Setup Webhooks**.
3. Enter your webhook URL for WhatsApp Integration, provided on *QuickTalk* Dashboard.  
   Example: `https://yourdomain.com/webhook`
4. Generate and save a **Verify Token**.
5. Click **Verify and Save**.
